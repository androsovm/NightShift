# NightShift

Automated overnight task runner for Claude Code. Loads tasks from a queue and executes them autonomously while you sleep.

## Dream Features

Функционал, о котором мы мечтаем. Однажды NightShift сможет всё это — и даже больше.

### Temporal Drift Engine
NightShift анализирует историю коммитов, предсказывает, какой код «устареет» через 3 месяца, и проактивно рефакторит его — пока вы ещё не знаете, что он сломается. Встроенная машина времени для кодовой базы.

### Dream Replay
Агент записывает «сны» — экспериментальные ветки, в которых он пробует радикальные архитектурные решения без последствий. Утром вы получаете дайджест: «Я попробовал переписать auth на Rust — стало в 12 раз быстрее, но потерялась совместимость с 3 плагинами. Вот diff, решай сам.»

### Codebase Whisperer
NightShift слушает ваш репозиторий как живой организм. Он чувствует «боль» в коде — участки с высокой когнитивной сложностью, забытые TODO, тесты, которые проходят случайно. Каждое утро — отчёт о самочувствии проекта с эмодзи-шкалой: от 😌 до 🔥.

### Parallel Universe PRs
Запуск N параллельных агентов, каждый из которых решает одну и ту же задачу разным способом. Утром вы выбираете лучший PR из «параллельных вселенных», а остальные тихо исчезают.

### Autonomous Dependency Negotiator
NightShift сам обновляет зависимости, но не просто `npm update` — он читает changelogs, анализирует breaking changes, мигрирует код и оставляет комментарии в стиле: «Обновил lodash. Заменил 14 вызовов _.get() на optional chaining. Сэкономил 47KB бандла. Не благодари.»

### Sleep-Driven Development (SDD)
Новая методология: вы описываете задачу перед сном, NightShift декомпозирует её, реализует, пишет тесты, деплоит на staging и собирает метрики. К утру — готовый PR с графиками производительности и предложением, что делать дальше. Ваш вклад — хороший сон.
